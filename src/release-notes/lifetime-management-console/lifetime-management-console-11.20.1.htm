LifeTime Management Console 11.20.1

Released on Jan 05, 2024

New in LifeTime Management Console 11.20.1

  • Site properties can now be defined as a secret, which will ensure their value is stored and encrypted in the database. This requires Platform Server 11.25 or higher and ServiceStudio 11.54.35 or higher. You can learn more about the feature here. (R11CT-2672)

Known issues

  • The installation checklist packaged with the installer is missing some relevant information for the scenarios of 'Upgrade to a new Major Release' and 'Upgrade to a new Release or Cumulative Patch'. The installation checklist available in the downloads page of this version does not have this problem.
