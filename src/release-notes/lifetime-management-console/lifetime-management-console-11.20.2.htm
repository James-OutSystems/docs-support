LifeTime Management Console 11.20.2

Released on Jan 24, 2024

Bug Fixing

  • Fixed a problem with the installation checklist where some specific parts about LifeTime were missing. (R11CT-2737)
  • Fixed an issue that was causing password fields to be hidden while creating Local admin profile with any of Legacy provider set as Authentication mechanism (RDOI-450)
  • Fixed an issue that was causing timeout during an environment synchronization process in LifeTime due to trash that was left behind in the database. This occurred during the update of the applications security settings at the final steps of the environment synchronization process, where some records were not being properly deleted after the update was done. (RPM-2308) Application Lifecycle LifeTime
