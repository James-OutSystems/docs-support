OutSystems Developer Cloud 2024-01-15

Released on Jan 15, 2024

New

  • Now on ODC Portal, on the log and trace detail pages, the metadata information about the source application now also shows which revision of the application generated that log/trace. (RDLOT-2480)

Bug Fixing

  • Fixed an issue with handling of multiple refreshes to the same aggregate or data action that caused the OnAfterFetch event handler to run as many times as the refreshes instead of just once for the last refresh. _(RDINC-17172)_ (RAR-2457)
  • Email screens are now being considered Application Objects. The platform was not considering these objects in the total AO count, which was not expected. This may impact the AO usage limit. (RDPSTT-1529)
