OutSystems Developer Cloud 2024-01-10

Released on Jan 10, 2024

New

  • Fixed an issue in the dependency analysis report during the deployment journey that could show in some cases a Ready with Warnings status even if there are no warnings in the report. (RDEL-1670)
  • From now on the input parameter mismatch in the Impact Analysis during the deploy operation will specify the name of each missing input parameter, instead of just the number of parameters. (RDEL-2029)
  • From now on the output parameter mismatch in the Impact Analysis during the deploy operation will specify the name of each missing output parameter, instead of just the number of parameters. (RDEL-2030)
  • From now on the structure attribute mismatch in the Impact Analysis during the deploy operation will specify the name of each missing attribute, instead of just the number of attributes. (RDEL-2031)
