ODC Studio 1.3.16

Released on Feb 12, 2024

New in ODC Studio 1.3.16

  • Users can now choose to show or hide entity attribute types in an entity diagram. Inspired by João Costa’s idea. (RDEV-6263)
  • Improved the experience of ODC Studio when opened with a session expired. Now instead of receiving an error, the user is guided to login in the browser. (RDEV-6273)
  • Improved the Satisfaction Feedback pop-up so that it is now the only place focused on overall satisfaction. Satisfaction was removed from Give Feedback and Satisfaction Feedback periodic pop-up is now more frequent. (RDOIBT-1903)

Bug Fixing

  • Fixed a silent error when clicking multiple times in the TAB key. (RDEV-6161)
  • Upgraded OpenSSL to version 3.0.13 (in debugger) to prevent a known vulnerability. CVSS score 5.3 (Medium). (RDEV-6157)
