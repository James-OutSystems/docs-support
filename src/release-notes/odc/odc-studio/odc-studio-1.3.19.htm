ODC Studio 1.3.19

Released on Mar 04, 2024

New in ODC Studio 1.3.19

  • Users can now choose to hide entity attributes in entity diagrams. (RDEV-6264)

Bug Fixing

  • Fixed an unexpected error that occurs when removing an If widget enclosing multiple widgets in a single branch. (RDEV-6347)
  • Fixed an issue that caused Find and Replace to not replace occurrences as expected. (RDEV-6358)
  • Fixed an unexpected error that occurred when starting a debug session while an aggregate is opening. (RDEV-6362)
  • Fixed an issue that caused the burger context menu to be displayed on the incorrect screen. This affected Windows users launching ODC Studio in a maximized state. (RDEV-6363)
