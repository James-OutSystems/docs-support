Integration Builder 1.50.0

Released on Mar 06, 2024

New in Integration Builder 1.50.0

  • It's now possible to set the Site path in SharePoint connections. With this, a single SharePoint instance, containing a version of the same site for each development stage, can be used to test your integrations. (RCNT-4806)
